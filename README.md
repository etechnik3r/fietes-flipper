# 🐸 Fietes Formenflipper: Die Pilz-Arena

Ein Flipper-Lernspiel für Kinder ab ca. 6 Jahren: **Buchstaben und
geometrische Formen kennenlernen** – beim Flippern in der Pilz-Arena.

## 🍄 Worum geht es?

Eine silberne Kugel saust durch die Pilz-Arena. Auf den **Pilz-Bumpern**
und den **Toren** stehen Buchstaben oder Formen (Kreis, Dreieck, Quadrat,
Stern). Trifft die Kugel einen davon, wird das Symbol **riesengroß
eingeblendet und laut vorgesprochen** – so prägt es sich ganz nebenbei ein.

Es gibt **kein Game Over und keinen Zeitdruck** – die Kugel kommt immer
wieder, und Punkte gehen nie verloren.

## 🕹️ So wird gespielt

- Am Anfang erscheint ein Titelbild – **einmal tippen** startet das Spiel.
- Jede Kugel wartet rechts in der **Abschuss-Gasse**. Ein Druck auf den
  großen roten **🚀-Knopf** schießt sie ins Feld – wie der Abzug beim
  echten Flipper.
- Unten am Bildschirm liegen **zwei riesige Knöpfe**: der **blaue** links,
  der **rote** rechts. Gedrückt halten hebt den jeweiligen Flipper-Arm,
  loslassen lässt ihn zurückfallen – so hält man die Kugel im Spiel.
- Jeder Treffer gibt Punkte: Pilze 🍄, Tore 🚪 und die goldenen
  Schleudern an den Seiten.
- Pro Runde gibt es **5 Kugeln** – der Vorrat steht oben in der Leiste
  (🎱). Sind alle gespielt, gibt es eine kleine **Feier mit Punkte-Bilanz**
  und die nächste Runde startet mit vollem Vorrat. Verlieren kann man nie.

## 🎯 Missionen

Immer wieder erscheint oben auf dem dunklen Schild eine **Mission**, zum
Beispiel *„Triff das M!“* oder *„Triff den Kreis!“*. Das passende Ziel
**leuchtet golden** auf dem Spielfeld. Wer es trifft, bekommt:

- 💯 **Bonuspunkte**
- ⭐ einen **Stern** (oben in der Leiste)
- 🎉 Konfetti und ein Lob

Die Mission wartet geduldig – Fehlschüsse sind völlig egal. Antippen des
Schilds liest die Mission noch einmal vor (praktisch für Nichtleser).

## 🪞 Der Spiegel-Zauber

Fällt die Kugel unten heraus, ist das keine Strafe, sondern ein
**Zauber-Moment**: Links steht eine **halbe Zeichnung** (Herz, Haus, Stern
oder Tannenbaum), in der Mitte die Spiegelachse. Mit dem Finger fährt das
Kind rechts über die Punktreihe und **zaubert so das Spiegelbild** herbei.
Ist das Bild komplett, gibt es Konfetti – und die nächste Kugel rollt an.

Wer mag, schaltet den Spiegel-Zauber im Menü aus und flippert einfach
sofort weiter.

## 🔘 Die Knöpfe im Überblick

| Knopf | Bedeutung |
|---|---|
| 🐸 (oben links) | Infos über das Spiel |
| 🎱 / 🔵 / ⭐ | Übrige Kugeln, Punkte und Missions-Sterne – antippen erklärt sie |
| ⚙️ | Menü: Buchstaben/Formen/Mix, Schwierigkeit, Spiegel-Zauber, Töne, Sprachausgabe |
| 🚀 (rechts unten) | Kugel abschießen, wenn sie in der Gasse wartet |
| Missions-Schild | Antippen liest die Mission vor 🔊 |
| Blauer/roter Knopf unten | Flipper-Arme bewegen |

Im Menü lässt sich einstellen, **was gelernt wird** (nur Buchstaben, nur
Formen oder bunt gemischt) und **wie schnell die Kugel** ist – drei
Schwierigkeiten: 🐢 **Leicht** (gemütlich, Standard), 🐇 **Mittel** und
🚀 **Schnell**. Punkte, Sterne und alle Einstellungen bleiben automatisch
gespeichert. Das **Zurücksetzen** ist mit einer kleinen Rechenaufgabe für
Erwachsene gesichert.

## ▶️ Starten

Einfach **`index.html`** im Browser öffnen – keine Installation nötig.
Als App: die Seite im Browser „Zum Startbildschirm hinzufügen“, dann läuft
das Spiel auch **offline**. Tipp: Sprachausgabe und Töne funktionieren am
zuverlässigsten über einen lokalen Webserver, z. B.
`python3 -m http.server 8000` → `http://localhost:8000`.
